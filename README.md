# Mr. Hopp's Playhouse 3D
A 3D mechanical translation of the 2D indie horror game Mr. Hopp's Playhouse. Built in Unity, this project adapts 2D stealth mechanics into a 3D environment.

## Features
- **Finite State Machine (FSM) AI:** Enemy stalker behavior using custom raycast-based vision cones.
- **3D Stealth Mechanics:** Vertical puzzle design replacing flat 2D logic.
- **Optimized Rendering:** Baked static lighting with culled real-time sources to maintain high FPS.

**Tech Stack:** Unity, C#
